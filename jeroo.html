Unit 1

In this unit we learned about the world of Jeroo and its environment. We learned how to hop(), toss(), plant(), pick(), turn(RIGHT),turn(LEFT), how to assign Jeroos a number of flowers and a starting point, as well as a starting direction to face.

Unit 2

In this unit we learned about Jeroo methods. We learned how to create a method, call a method from method main(), call a method from within another method, and use methods to solve complicated problems quickly.

Unit 3

In this unit we learned about Jeroo loops. We learned how to create a while() loop, use loops inside loops, use loops to solve problems that involve doing the same thing again and again.

Unit 4

In this unit we learned about if statements. We learned how to create an if() statement, integrate an else() statement inside the if() statement, integrate and else if() statement inside the if() statement and use if() statements inside loops to look for a certain object/landscape over a large area.
