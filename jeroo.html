Unit 1

Description

In this unit we learned about the world of Jeroo and its environment. We learned how to move, dodge nets, keep away from the water and stop when we cannot. We learned how to keep our Jeroo safe and eliminate danger(nets) by tossing flowers.

Concepts Learned:

  • Move Jeroo: hop();, hop(x);
  • How to manage flowers: toss();, plant();, pick();
  • How to use flowers to eliminate nets.
  • Turn Jeroo: turn(RIGHT);,turn(LEFT);
  • Create a new Jeroo: Jeroo x = new Jeroo();
  • How to assign Jeroos a number of flowers and a starting point, as well as a starting direction to face: Jeroo x = new Jeroo(x,y,DIRECTION,FLOWERS);
  • Start the main method: method main();

Unit 2

Description

In this unit we learned about Jeroo methods. We learned how to use methods right from the main method by assigning it to a certain Jeroo, or calling it from another method, having to assign only one method to a Jeroo. We used methods as a shortcut to solve problems that have long parts that show up again and again.

Concepts Learned:

  • Create a Jeroo method: method x()
  • How to call a method from within another method: x();
  • How to call a method from the main method: Jeroo.x();
  • Use methods to solve problems that have certain parts that show up many times.

Unit 3

Description

In this unit we learned about Jeroo loops. We learned how to use loops inside methods and also from the main method. We used loops as a shortcut to solve problems that have parts that repeat again and again in the same order.

Concepts Learned:

  • Create a while loop: while()
  • How to use a loop from within a method: while(isWater());
  • How to use a loop from within the main method: while(Jeroo.isWater());
  • Use loops to solve problems that have certain parts repeat in a certain order.
  • Use boolean methods inside a while() loop: while(isWater(AHEAD) || !isWater(RIGHT))

Unit 4

Description

In this unit we learned about if statements. We learned how to use if statements inside methods and also from the main method. We learned how to use boolean statements inside the if statements. We also learned how to integrade else if() statements as well as else statements. We used all of those statements to solve many problems, suck as protecting the Jeroo by causing it to do something whenever there is a net AHEAD.

Concepts Learned:

  • Create an if statement: if()
  • How to use an if statement from within a method: if(isWater());
  • How to use an if statement from within the main method: if(Jeroo.isWater());
  • Use if statements to sense danger and prevent the Jeroo from getting hurt.
  • Solve tasks that instruct the Jeroo to do a certain thing every time it senses something.
  • Integrate else() and else if() statements if the Jeroo needs to do different things in different cases.
